    Class MediaSource

    Base class for media data source components producing some media frames

    This component encapsulates one of 4 types of media tracks, depending on the actual class deriving from it:

    • AudioSender : produces audio frames locally, to be sent to the remote peer
    • AudioReceiver : receives audio frames from the remote peer
    • VideoSender : produces video frames locally, to be sent to the remote peer
    • VideoReceiver : receives video frames from the remote peer

    The media source can be conveniently played using a MediaPlayer.
