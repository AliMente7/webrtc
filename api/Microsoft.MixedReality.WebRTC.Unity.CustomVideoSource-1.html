    Remarks

    This must comply with the 'msid' attribute rules as defined in https://tools.ietf.org/html/draft-ietf-mmusic-msid-05#section-2, which in particular constraints the set of allows characters to those allowed for a 'token' element as specified in https://tools.ietf.org/html/rfc4566#page-43:

    • Symbols [!#$%'*+-.^_`{|}~] and ampersand &
    • Alphanumerical [A-Za-z0-9]
